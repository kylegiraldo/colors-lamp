# colors-lamp

The colors application allows users to add colors to a database and search for those colors.

Technologies used for this project include:
Visual Studio Code
FileZilla
Postman

Setup:
Create a database with both Users and Colors tables
use the following fields for Users: ID(Primary Key), FirstName, LastName, Phone, Email, UserID
use the following fields for Colors: ID(Primary Key), Name, UserID
Create the api endpoints using AddColor, Login, and SearchColors
Upload all the frontend files to your web page using FileZilla

This README assumes that you already have a domain setup
